# Glassy Portfolio - "iOS 26" Style

This is a portfolio website built with React, Vite, and Tailwind CSS, featuring a futuristic "iOS 26" glassmorphism aesthetic.

## Features
- **Glassmorphism UI**: Heavy blur, translucency, and vibrant backgrounds.
- **Dynamic Animations**: Powered by Framer Motion.
- **Responsive Layout**: Works on mobile and desktop.
- **Components**:
  - `GlassCard`: Reusable glass container.
  - `Dock`: macOS/iOS style floating navigation.
  - `Hero`, `About`, `Experience`, `Contact` sections.

## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Run Development Server**:
    ```bash
    npm run dev
    ```

3.  **Build for Production**:
    ```bash
    npm run build
    ```

## Customization
- Edit `src/sections` to update content.
- Modify `src/index.css` to tweak glass styles.
